What is grid based design?

It is a webpage that is designed to work in and around the constraints of a grid system (a series of intersecting horizontal and vertical lines). It creates order among elements in a website.

Whats all the hype about responsive webpages?

It is the only way to make websites available to all users no matter which type of device is used to view the content. Mobile browsing use is rapidly growing and could one day overtake internet browsing.

What is semantic structure?

In HTML it is the tags we use to mark a document up. Creating semantic markup is nothing more complicated than understanding the language of HTML and using it to create a document that has semantic meaning.
